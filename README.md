🛠️ СТОЛ И ТОЧКА

Философия проста: положил ПК на стол, вставил флешку, записал — И ТОЧКА.

Это утилита для тех, кому нужно быстро и без лишних вопросов подготовить загрузочную флешку с образом (например, PearOS). Никаких облаков, никаких сложных мануалов. Чисто железо и софт.
🔌 Как это работает:

    КИДАЕШЬ образ в программу.

    ТЫКАЕШЬ флешку в порт.

    ЖМЕШЬ «СТАРТ».

    ГОТОВО.

⚙️ Технические характеристики:

    Совместимость: GPT (UEFI) для новых, MBR (BIOS) для дедовских сборок.

    Надежность: Режим DD для тех случаев, когда обычный ISO не хочет грузиться.

    Проверка: Можно прогнать на бэд-блоки, если флешка подозрительно дешевая.

⚠️ Предупреждение:

Нажал СТАРТ — данные на флешке улетели в стратосферу. Без обид.
